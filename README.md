# Hubot: hubot-tahoe


See [`src/tahoe.coffee`](src/tahoe.coffee) for full documentation.

## Installation

Add **hubot-tahoe** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-tahoe": ">= 0.0.0"
}
```

Add **hubot-tahoe** to your `external-scripts.json`:

```json
["hubot-tahoe"]
```

Run `npm install hubot-tahoe`
